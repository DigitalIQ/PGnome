# PGNome: Isometric Evolution

PGNome is an isometric browser game focused on genetics, evolution, and autonomous creature management. Acting as an overarching observer, you manage an ever-changing world to encourage your creatures to thrive, find mates, and expand their genetic pool.

## Features

- **Isometric World Generation**: Procedurally generated biomes including Water, Sand, Meadow, Mud, Forest, and Rock.
- **Autonomous Creatures**: Beings that act on their own needs—hunger, energy, happiness, and arousal. They seek suitable environments based on their unique genetics.
- **Genetic System**: Creatures inherit traits from their parents (color, size, metabolism, preferred climate, sociability) with slight mutations.
- **Dynamic Environment**: The world evolves over time, shifting biomes to simulate long-term environmental changes.
- **God-Mode Interactions**:
  - Drag and drop creatures to relocate them.
  - Terraform tiles to manually sculpt the perfect environment.
  - Drop food to lure and sustain creatures.
  - Inspect tiles and creatures to analyze their genetics and current states.
- **No Dependencies**: Built entirely with Vanilla JavaScript, HTML, and CSS (Canvas API).

## How to Play

Since this project has zero external dependencies, simply open the `index.html` file in any modern web browser to start playing immediately. No build step or local web server is required.
